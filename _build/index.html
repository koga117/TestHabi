API de Inmuebles

Introducción

La API de Inmuebles permite consultar y gestionar inmuebles disponibles para la venta. Esta API ofrece una serie de recursos que te permiten:

  • Obtener una lista de inmuebles.

  • Filtrar los inmuebles por año de construcción, ciudad y estado.

  • Obtener información detallada de un inmueble específico.

  • Crear un nuevo inmueble.

  • Actualizar los datos de un inmueble existente.

  • Eliminar un inmueble.

Recursos

  • Guía de inicio rápido: http://localhost:5000/inmuebles

  • Referencia de la API: http://localhost:5000/inmuebles
